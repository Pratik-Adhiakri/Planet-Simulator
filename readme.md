<h1>Planet Simulator</h1>
<p>Planet Simulator is a simulator that simulates different heavenly bodies's gravitational force and orbits with real time physics. It is made by using HTML/CSS/JS. </p>

<h3>Technologies Used</h3>
<p>1/ HTML-HTML is used for the basic structure and to add the canvas for the game to work.</p>
<p>2. CSS-CSS is used to style the different elements and make the UI attractive as well as interactive.</p>
<p>3. JS-JS is the main part of this and is used to input the real time phscics and make the HTML Canvas working again. </p>

<h3>Controls</h3>
<p>1. Scroll- Scroll is used to zoom in or out, it is very basic.</p>
<p>2. Right Click + drag - It is used to rotate the camera to go wherever we want.</p>
<p>3. Left Click + drag - It is used to spawn a heavenly body and drag can be used to make orbits and set its direction.</p>

<h3>How to run it locally?</h3>
<p>1.Download VS Code and fork this repository onto your github.</p>
<p>2.After downloading VS Code, git clone this repository into your device and select the destination.</p>
<p>3.Now, go to extension and download the Live Server extension.<p>
<p>4.Click the 'Go Live' button in your bottom right corner of the screen.</p>
<p>5. ENJOY!!!</p>

<h3>How to run in browser?</h3>
<p>1. Install any web browser in your respective device.</p>
<p>2. Open https://pratik-planetsimulator.netlify.app in your browser.</p>
<p>3. Enjoy!!!</p>
